# Dados sobre comportamento extrovertido vs. introvertido
Sobre o conjunto de dados
Visão geral

Mergulhe no conjunto de dados de traços de personalidade extrovertidos vs. introvertidos, uma rica coleção de dados comportamentais e sociais projetada para explorar o espectro da personalidade humana. Este conjunto de dados captura indicadores-chave de extroversão e introversão, tornando-se um recurso valioso para psicólogos, cientistas de dados e pesquisadores que estudam comportamento social, previsão de personalidade ou técnicas de pré-processamento de dados.

Contexto

Traços de personalidade como extroversão e introversão moldam a forma como os indivíduos interagem com seus ambientes sociais. Este conjunto de dados fornece insights sobre comportamentos como tempo gasto sozinho, participação em eventos sociais e engajamento em mídias sociais, possibilitando aplicações em psicologia, sociologia, marketing e aprendizado de máquina. Seja para prever tipos de personalidade ou analisar padrões sociais, este conjunto de dados é sua porta de entrada para descobrir insights fascinantes.

Detalhes do conjunto de dados

Tamanho : O conjunto de dados contém 2.900 linhas e 8 colunas.
